Для владельцев Smart TV (Samsung, LG)

Если ваш телевизор не на базе Android, вы всё равно можете смотреть StreamLume TV без дополнительных приставок!

  1. Установите бесплатное приложение Media Station X из магазина приложений вашего ТВ.
  2. Запустите его, зайдите в Settings (Настройки) ➡️ Start Parameter (Стартовый параметр) ➡️ Setup.
  3. Введите адрес нашего сервера: streamlume-tv-svmorozoww.amvera.io
  4. Нажмите галочку (Enter) и выберите Yes для перезапуска.

✨ Готово! Плеер загрузится автоматически и будет мгновенно реагировать на ваш пульт.
